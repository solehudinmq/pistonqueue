1. running data on the main topic stuck in redis via systemd :
- create a new service : 
  a. open terminal
  b. cd /etc/systemd/system
  c. which bundler # bundler-installation-location
  d. which ruby # ruby-installation-location
  e. sudo nano pending-consumer.service
  f. Fill in 'pending-consumer.service' according to the file 'example/services/pending-consumer.service'
  g. sudo systemctl daemon-reload
  h. sudo systemctl enable pending-consumer.service
  i. sudo systemctl start pending-consumer.service
  j. sudo systemctl status pending-consumer.service
  k. sudo journalctl -u pending-consumer.service -f

- restart service if there are changes in the file :
  a. open terminal
  b. cd /etc/systemd/system
  c. sudo nano pending-consumer.service
  d. add changes in 'pending-consumer.service' file.
  e. sudo systemctl daemon-reload
  f. sudo systemctl restart pending-consumer.service
  g. sudo systemctl status pending-consumer.service
  h. sudo journalctl -u pending-consumer.service

- delete service :
  a. open terminal
  b. cd /etc/systemd/system
  c. sudo systemctl stop pending-consumer.service
  d. sudo systemctl disable pending-consumer.service
  e. sudo rm -rf pending-consumer.service
