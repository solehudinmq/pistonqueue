1. run consumer for light i/o bound tasks in systemd : 
- create a new service : 
  a. open terminal
  b. cd /etc/systemd/system
  c. which bundler # bundler-installation-location
  d. which ruby # ruby-installation-location
  e. sudo nano light-io-bound-consumer.service
  f. Fill in 'light-io-bound-consumer.service' according to the file 'example/services/light-io-bound-consumer.service'
  g. sudo systemctl daemon-reload
  h. sudo systemctl enable light-io-bound-consumer.service
  i. sudo systemctl start light-io-bound-consumer.service
  j. sudo systemctl status light-io-bound-consumer.service
  k. sudo journalctl -u light-io-bound-consumer.service -f

- restart service if there are changes in the file :
  a. open terminal
  b. cd /etc/systemd/system
  c. sudo nano light-io-bound-consumer.service
  d. add changes in 'light-io-bound-consumer.service' file.
  e. sudo systemctl daemon-reload
  f. sudo systemctl restart light-io-bound-consumer.service
  g. sudo systemctl status light-io-bound-consumer.service
  h. sudo journalctl -u light-io-bound-consumer.service -f

- delete service :
  a. open terminal
  b. cd /etc/systemd/system
  c. sudo systemctl stop light-io-bound-consumer.service
  d. sudo systemctl disable light-io-bound-consumer.service
  e. sudo rm -rf light-io-bound-consumer.service
